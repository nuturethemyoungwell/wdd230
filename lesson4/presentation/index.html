CHROME DEVELOPER TOOLS PRESENTATION

With this presentation, we are going to talk briefly about the following:

  • Meaning of Chrome DevTools
  • Features of Chrome DevTools
  • Accessing the DevTools
Meaning of Chrome DevTools

As the name implies, these are tools which are built inside Chrome browser. Chrome DevTools are very useful tools which enable web developers to see the result of their source codes, and also allows them work directly in the browser, preview HTML and style changes, console JavaScript codes and, or allows for debugging.

It is a tool which allows developers to analyze the contents on their web page.

Features of the Devtools

    The following snippets are the characteristics or uses of Devtools

  • Inspects web pages
  • Modifies HTML and CSS directly on the webpage
  • Debugging Javascripts directly on the web page
  • Imitates different viewports
  • Measures page performance

How to access Devtools:

    Devtools can be found directly in the chrome browser.
    To access these tools;

    1. Right-click any area on the web page and select "Inspect". That is the last item on the pop up menu.

    2. You can also select view ("the three dots at the top right corner of your page") > Developer > Developer tools.

Available Tabs:
    • Element Tab: Is the tab which display any changes rendered /altered to the inner HTML text or content through Javascript.
      Function: Its function enables you to edit the code you are inspecting; since, it shows the error causing issues for the page by underlining it with a red zigzag line. That is, the line of code showing the error is underlined.

    • Console Tab: This tab automatically detects errors within the web page. It highlight why the erroType is showing. It helps trace the issue to that line of code.

      For example;
      Uncaught TypeError: Usually, this indicate there was TypeError with an HTMl element on the page. Typically, it suggest there is faulty code within a javascript function.

      Uncaught SyntaxError: This indicate there was a syntax error with an HTML element on the page. Actually, this also suggest your code has used inappropriate characters like (<"/*^#).

    • Resources Tab: This tab displays the various files loaded for the page you are inspecting. The resource tab contains either files or folders of HTML contents, styles files, javascript contents and etc.

    • Network Tab: This is the tab which enables developers to see and analyze the network requests that makes up the page loaded for a particular section. It can be used to activate or set the connectivity status of the web page at a particular section. It also help determine the performance of a web page.

    • Memory Tab: Shows the HTTP traffic and memory usage over time.
